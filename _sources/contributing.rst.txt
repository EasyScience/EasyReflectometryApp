==================================
Contributing to this documentation
==================================

This documentation lives in the `EasyReflectometryApp`_ repository, in the :code:`docs` directory. 
So, if you want to contribute to this documentation, you can do so by opening a pull request there. 
To check locally that the changes do not break anything, you can run the following. 

.. code-block:: console

    $ cd docs
    $ make html

This will build the html documentation in the :code:`docs/_build` directory. 

.. _EasyReflectometryApp: https://github.com/easyScience/EasyReflectometryApp
